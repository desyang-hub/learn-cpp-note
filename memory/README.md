## 内存管理

### 堆栈

1. 栈：由编译器自动管理，用于存储局部变量，函数参数，函数返回值等。分配/释放速度快，空间有限只有几MB, 变量离开作用域自动释放
2. 堆：由程序员手动管理，用于存储动态分配的内存(new/delete)，空间大，但是分配/释放速度慢，容易出现内存泄漏和野指针异常。

面试常问：

1. 栈和堆的区别（生命周期、大小、分配速度、管理方式）
栈：生命周期由编译器管理，自动分配和释放，空间有限，分配速度快，管理方式是先进后出。
堆：由程序员手动管理，生命周期由程序员控制，空间大，分配速度慢。

2. 为什么RAII很重要？给个例子说明没有RAII可能造成的问题。
RAII（Resource Acquisition Is Initialization）是一种资源管理技术，通过将资源的生命周期与对象的生命周期绑定，确保资源在使用完毕后自动释放，避免资源泄漏和内存泄漏等问题。例如，使用智能指针（如unique_ptr、shared_ptr）管理动态分配的内存，当智能指针对象被销毁时，会自动释放所管理的内存，避免内存泄漏。
