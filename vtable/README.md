## 虚函数表

1. 虚函数表在内存中的布局？如果多个继承呢？
虚函数表在内存中的布局是：虚函数表指针 -> 虚函数表 -> 虚函数表中的函数指针 -> 函数体。
如果多个继承，那么每个类都有自己的虚函数表，虚函数表指针指向自己的虚函数表。


2. 构造函数能否是虚函数？析构函数为什么通常要声明为虚函数？
构造函数不能是虚函数，因为虚函数表是在对象创建时初始化的，而构造函数是在对象创建之前调用的，所以构造函数不能是虚函数。

析构函数通常要声明为虚函数，因为虚函数表是在对象创建时初始化的，而析构函数是在对象销毁时调用的，如果析构函数不是虚函数，那么在多态情况下，子类的析构函数不会被调用，导致内存泄漏。

3. 纯虚函数和抽象类。
纯虚函数是只有声明没有实现的虚函数，抽象类是包含至少一个纯虚函数的类。抽象类不能被实例化，只能被继承。
