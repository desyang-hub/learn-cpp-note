## 移动语义


### 面试常问

1. 左值和右值的区别？什么是将亡值（xvalue）？
左值是指有变量名的值，右值是临时值，没有变量名。将亡值是即将被销毁的左值。

2. 移动构造和拷贝构造的区别？什么时候会自动生成移动操作？
移动构造作用的目标是一个右值引用，或是被标记为将亡值的左值引用。移动构造函数的参数类型是右值引用，而拷贝构造函数的参数类型是左值引用。当对象被移动时，会自动生成移动构造函数。


3. 为什么移动后要把原对象置为可析构状态？
移动语义的本意是将资源所有权从一个对象转移到另一个对象，因此需要将原对象置为可析构状态，以避免重复析构。
